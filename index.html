1
Open Windows Terminal as Admin
Press Win+X, select "Terminal (Admin)"
Or search "Terminal" in Start menu, right-click, "Run as administrator"
2
Paste this command and press Enter
irm https://raw.githubusercontent.com/marcusash/gh-copilot-setup/main/bootstrap.ps1 | iex
Script installs Git, VS Code, PowerShell 7, and sets up both agent workspaces.
3
Launch agents from desktop shortcuts
After install completes, you'll have two shortcuts on your desktop: AI Maker and AI Workbench (both with GitHub logo). Click to launch each GitHub CLI workspace.
4
Initialize AI Maker first
Open AI Maker. In the GitHub CLI, say: Good morning. AI Maker runs the onboarding interview. Answer one question at a time.
5
Initialize AI Workbench
Open AI Workbench. In the CLI, say: AI Workbench ready. What are we working on? No onboarding interview. It learns through the work.

Choose your agent based on the work
Use AI Maker for day-to-day work. Use AI Workbench for technical investigation, research, debugging, or building. Each has its own CLI workspace. Each specializes.
